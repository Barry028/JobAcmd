設計規範

東柏資訊網頁設計模版，設計規範文件，使用 SCSS 開發，資料夾主要分成 7 個 
                                
                                    scss/
                                      ├── animations/ --> 動畫（基本上不動）
                                      ├── base/ --> Css Reset 基本標籤設定
                                      ├── components/ --> Components 頁面元件樣式
                                      ├── layout/ --> 佈局以及全域的樣式設定
                                      ├── mixins/ --> 存放所有 Mixin 的撰寫
                                      ├── pages/ --> 個別的頁面請放在此資料夾內
                                      ├── helpers/ --> 網站變數及一些事件的設定

  • 可以在相對應的資料夾內新增 _xxxx.scss 檔案，再去 main.scss 檔案內將所新增的 .scss 檔案引入。
  • 此專案使用 npm gulp 編譯 sass/scss。
  • 檔名前加上 _ 則該檔案不會被轉出編譯成 .css 檔案。
  • 編譯有順序性，因此需先載入變數、事件等等必要檔案後在做，詳細請參考 main.scss 。
